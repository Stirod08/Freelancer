Tecnologías y Otros

HTML5

HTML 5 es la última versión del lenguaje de marcado utilizado para crear páginas web. Proporciona nuevas etiquetas y atributos, mejoras en la semántica, soporte para multimedia y mejoras en la accesibilidad.

CSS3

CSS 3 es la última versión del lenguaje de hojas de estilo utilizado para dar estilo a las páginas web creadas con HTML. Proporciona nuevas propiedades, mejoras en la capacidad de diseño responsivo y soporte para animaciones y efectos avanzados.

VS Code

VS Code es un editor de código fuente gratuito y de código abierto desarrollado por Microsoft. Proporciona una amplia gama de características y extensiones para aumentar la productividad de los desarrolladores, incluyendo resaltado de sintaxis, depuración, control de versiones y más.
