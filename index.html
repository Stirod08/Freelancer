Tecnologías y Otros

HTML5

HTML 5 es un lenguaje de marcado para crear páginas web, con nuevas etiquetas y atributos, mejoras en la semántica, soporte multimedia y accesibilidad.

CSS3

CSS 3 es un lenguaje de hojas de estilo para dar estilo a páginas web creadas con HTML, con nuevas propiedades, diseño responsivo y soporte para animaciones y efectos avanzados.

VS Code

VS Code es un editor gratuito de código abierto de Microsoft, con características y extensiones para aumentar la productividad de los desarrolladores, como resaltado de sintaxis, depuración y control de versiones.
